Local Storage in javascript

Local storage allows to store data in the browser. Data is stored in key/value pairs. It is specific to domain only. The localStorage object stores data with no expiration date. The data is not deleted when the browser is closed, and are available for future sessions.

It has a capacity of upto 10 mb, which is more that cookies and session storage. The data stored in the local storage remains there until manually deleted, it has no expiry date

You can retrieve the data in the same browser only, data stored in a browser locally cant be obtained if we open the site in another browser.

Syntax

To access local storage property:

window.localStorage;

but since its part of window object we can simply use:

localStorage;

To clear the content:

localStorage.clear();

To add a key value pair to the local storage object:

localStorage.setItem("key","value");

To delete a key value pair from the local storage:

localStorage.removeItem("key");

To get a value when you pass in a key:

localStorage.getItem("key");
